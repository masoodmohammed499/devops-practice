## Objective
Connect to an AWS EC2 Ubuntu instance using SSH key-based authentication.

## Steps Performed
- Launched Ubuntu EC2 instance
- Configured Security Group to allow SSH (port 22)
- Connected to EC2 using SSH key (.pem)
- Verified login using whoami and hostname
- Verified public IP using curl ifconfig.me
