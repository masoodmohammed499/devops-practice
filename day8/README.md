# Day 8 - EC2 + IAM Role + S3

**Objective:** Launch an EC2 instance and securely access S3 using IAM Role without any credentials.

**Steps Done:**
1. Launched Ubuntu EC2 instance
2. Created IAM Role with AmazonS3FullAccess
3. Attached IAM Role to EC2
4. Verified access to S3 using `aws s3 ls`
5. SSH into EC2 using key pair
